These are a set of ANSI C++ Library Routines that make up the 
MTC++ Class Library.

Some of the classes include:

MTCstring   - a character string class.
MTCinitFile - used to access a Microsoft Windows (tm) .ini file format.
errormsg    - a set of user-interface independent messaging routines.
MTCBST      - a Binary Search Tree Container Class Template. 
MTCRBT      - a Red-Black (balanced) Search Tree Container Class Template. 
MTClist     - a linked list container class template.
MTCraof     - Random Access Object Format -- hash index of file.
